📋 学生岗位实习行动指南

化学工程系 · 2024级 · 2026年*月*日 — 2027年6月25日

一、选择你的实习类型

区内集中实习

由系部统一安排实习单位，统一组织管理。实习开始前10日签订三方协议，按市域分开提交材料。

你需要完成的材料

  • 实习申请 — i鄂院-智慧实训安全平台-岗位实习（需要先提交企业实习申请，审批后方可进行后续操作）
  • 附件3：三方协议 — 学校统一组织签订，一式三份（甲、乙、丙三方各执一份），逐空填写、手写签署、加盖单位公章，不得删减修改
  • 三方协议扫描件 — 签订后提交
  • 附件5、6：岗位实习方案及任务书 — 各专业对应方案（见下方专业要求），每人1份，作为实习培训内容
  • 实习周志 — 每周1篇，内容包含岗位操作、工艺学习、问题反思、安全心得（前半年18篇，后半年16篇）
  • 实习总结报告 — 实习结束时提交（各专业字数要求不同，见下方）
  • 企业实习鉴定表 — 企业指导师傅/企业负责人填写评价并加盖公章
  • 岗位实习证明 — 企业指导师傅/企业负责人填写评价并加盖公章（每半年提交一次）

区内自主实习

自行选择实习单位，需经审核备案后方可实习。自行选择的实习单位需经审核、备案。

你需要完成的材料

  • 实习申请 — i鄂院-智慧实训安全平台-岗位实习（需要先提交企业实习申请，审批后方可进行后续操作）
  • 附件2：实习单位考察报告一览表 依据营业执照完成，完成后发给班主任
  • 附件3：三方协议 — 签订后提交，一式三份，逐空填写、手写签署、加盖单位公章
  • 三方协议扫描件 — 签订后提交
  • 附件5、6：岗位实习方案及任务书 — 各专业对应，每人1份
  • 附件7：学生自选单位岗位实习审批表 — 一式四份（本人、企业、二级学院、教务处各一份），实习开始前提交审批
  • 附件8：自主实习家长知情书 — 家长（法定监护人）手写签署，声明知情并自行负责子女实习期间的安全问题，如发生任何意外或事故，自行承担全部责任
  • 附件9：单独居住家长知情书 — 如不在实习单位统一安排的宿舍住宿则需提交，家长签署后自行负责在外居住期间的安全（包括交通安全）
  • 附件11：学生实习管理制度 —需要企业盖章
  • 附件12：学生实习安全管理规定 — 需要企业盖章
  • 实习周志 — 每周1篇，内容包含岗位操作、工艺学习、问题反思、安全心得（前半年18篇，后半年16篇）
  • 实习总结报告 — 实习结束时提交（各专业字数要求见下方）
  • 企业实习鉴定表 — 企业指导师傅/企业负责人填写评价并加盖公章
  • 岗位实习证明 — 企业指导师傅/企业负责人填写评价并加盖公章（每半年提交一次）
⚠ 自主实习学生必须保证有企业人员向班主任"日报告"实习情况。需向系里提供用人单位的接收函及岗位实习证明，并办理相关手续方可离校。

跨省实习

实习单位在内蒙古自治区以外的省份。需于五月底前提出申请，经院长办公会审议通过后方可组织签订三方协议。备案材料于实习开始前15日提交，至少提前10日向市、省两级主管部门备案。

你需要完成的材料

  • 实习申请 — i鄂院-智慧实训安全平台-岗位实习（需要先提交企业实习申请，审批后方可进行后续操作）
  • 附件2：职业学校学生跨省实习备案表 — "一省一表"，分省填报。认真填写表中每一项信息，多专业/多单位/多时间需标清匹配关系。各级部门需给出备案意见并盖章
  • 附件4：跨省实习保单明细 — 提供为实习学生和校方指导教师购买的实习责任保险、意外伤害险等保单原件和明细，加盖学校公章，保险有效期须覆盖实习全过程。如有补购保险安排需提供相关说明并盖章
  • 附件5：学生跨省实习情况统计表 — "一省一表"，以"内蒙古赴X省实习备案X人"命名。除广东、河北外用此表；赴广东用附件6，赴河北用附件7
  • 附件3：三方协议 — 一式三份，逐空填写、手写签署、加盖单位公章
  • 三方协议扫描件 — 签订后提交
  • 家长知情同意书 — 作为三方协议附件，家长（法定监护人）签署
  • 附件5、6：岗位实习方案及任务书 — 各专业对应，每人1份
  • 实习周志 — 每周1篇，内容包含岗位操作、工艺学习、问题反思、安全心得（前半年18篇，后半年16篇）
  • 实习总结报告 — 实习结束时提交
  • 企业实习鉴定表 — 企业指导师傅/企业负责人填写评价并加盖公章
  • 岗位实习证明 — 企业指导师傅/企业负责人填写评价并加盖公章（每半年提交一次）
⚠ 关键时间：五月底前申请 → 审议通过后签协议 → 实习开始前15日备案 → 至少提前10日报备。
⚠ 备案表、三方协议、统计表、家长知情同意书上的实习起止时间必须一致，精确到日。实习单位名称须与企业公章一致。如有备案修正要求，5日内补正上报完毕。

二、选择你的专业

对应文件附件5（岗位实习方案）、附件6（岗位实习任务书）
实习标准依据教育部《分析检验技术专业岗位实习标准》
实习岗位方向检验检测相关岗位。常用仪器：紫外可见分光光度计、气相色谱仪、液相色谱仪、原子吸收分光光度计、天平、酸度计等。需掌握基本操作、日常维护与简单故障排查。
实习周志每周1篇，内容包含岗位操作、工艺学习、问题反思、安全心得。
实习总结报告不少于3000字。包含：实习企业概况、实习操作技能、岗位操作内容、技能收获、行业认知及职业规划。
时间安排第1-2周入职熟悉 → 第3-4周基础实操 → 第5-8周常规检测 → 第9-12周仪器操作 → 第13-16周综合工作 → 第17-18周总结提升
考核方式安全考核30分 ＋ 实操考核40分 ＋ 材料考核20分 ＋ 考勤素养10分。安全考核出现违规操作直接不及格。企业师傅对实操打分。
对应文件附件5（岗位实习方案）、附件6（岗位实习任务书）
实习标准依据教育部《高等职业学校化工技术类专业岗位实习标准》《安全技术与管理专业岗位实习标准》
实习岗位方向化工操作岗、分析检验岗、各类企业安全管理岗。
实习任务①了解企业文化和管理制度；②熟悉安全生产规章制度、操作规程和应急预案，完成三级安全教育；③学习实习车间原材料及产品的性质、储运方式、反应原理、工艺流程、工艺参数；④掌握危险化学品泄漏、火灾、爆炸等事故的应急处置方法；⑤学习岗位操作规程和工艺流程图；⑥学习安全生产法律法规，应用危险源辨识方法进行安全风险排查和隐患记录；⑦学习原料、中间品、产品及水质相关分析检测方法。
实习周志每周按时提交1篇岗位实习周志。
实习总结报告实习结束时提交总结报告一篇。另需提交实习证明1份、实习鉴定表1份（企业审核评价，签字盖章）。
时间安排第三学年第五、第六学期。2026年9月1日 — 2027年6月25日。
考核方式企业评定（实习过程表现）50% ＋ 周志（每周1篇）10% ＋ 学校指导老师评价20% ＋ 实习报告20%
对应文件附件5（岗位实习方案）、附件6（岗位实习任务书）
实习标准依据教育部《应用化工技术专业岗位实习标准》
实习岗位方向化工生产相关岗位。掌握化工单元操作、工艺流程、常用设备基础理论，能识别化工设备、仪表，读懂工艺图纸与操作规程。熟悉化工原料、中间体、成品基本理化性质。
实习周志每周实习周记，总结本周技能收获、知识感悟、岗位思考，共计不少于规定篇数。
实习总结报告不少于1500字。包含：实习概况、岗位介绍、工艺流程、实操内容、问题分析、心得体会、职业规划，可附带工艺简图、合理化建议等。
时间安排大三学年第五、第六学期，共计34周。
考核方式岗位实操50%（企业师傅评价）＋ 出勤纪律20%（校内指导教师评价）＋ 实习材料30%（周记10%＋报告20%）。等级评定：优秀≥90、良好≥76、合格≥60、不合格＜60。如有技术改进或革新等可作为加分项，需有证明材料，最高加20分。抄袭、空缺酌情扣分。
对应文件附件5（岗位实习方案）、附件6（岗位实习任务书）
实习标准依据教育部《煤化工技术专业岗位实习标准》
实习岗位方向煤气化、气体净化、煤制合成、煤制燃料、煤制化学品等全流程生产岗位，覆盖内操、外操在岗作业人员。
外操岗：生产现场巡回检查、设备启停状态确认、手动阀门操作、现场物料采样、机泵及管线日常维护配合、跑冒滴漏排查、现场突发情况应急处置等。
内操岗：DCS中控系统实时监控、工艺参数调整、安全联锁管理、生产负荷调控、下达岗位操作指令、研判工艺异常工况等。
实习任务安全教育任务（三级安全培训、应急演练、安全学习总结）、工艺学习任务（每周整理工段工艺笔记）、现场实操任务（定时巡检、数据记录、设备润滑、物料取样等辅助作业）、文档整理任务（运行日志、交接班记录、实习周记、月度小结）、合规纪律任务（保密规定、禁烟、动火作业等现场管理规定）。
实习周志每周1篇，内容包含岗位操作、工艺学习、问题反思、安全心得。另需按月整理月度实习小结。
实习总结报告不少于3000字。包含：实习企业概况、实习工段工艺流程、岗位操作内容、技能收获、行业认知及职业规划。
时间安排第1周安全集训 → 第2-5周岗位认知 → 第6-34周岗位实习（全程在师傅监护下开展）。
考核方式安全考核30分 ＋ 实操考核40分（企业师傅打分，独立完成巡检记录、设备点检为达标）＋ 材料考核20分 ＋ 考勤素养10分。违规操作安全考核直接判为不及格。
对应文件附件5（岗位实习方案）、附件6（岗位实习任务书）
实习标准依据教育部《煤炭清洁利用技术专业岗位实习标准》
实习岗位方向煤炭洗选、煤气化、煤气净化、合成精制及其他煤炭深加工生产岗位，覆盖装置全部内操、外操在岗作业人员。
外操岗：现场巡检、设备启停确认、阀门手动操作、现场采样、机泵维护配合、管线查漏、现场应急处置等。
内操岗：DCS中控监视、工艺参数调节、联锁管控、生产负荷调整、操作指令下达、工艺异常研判等。
实习任务安全教育任务（三级安全培训、应急演练并提交学习总结）、工艺学习任务（每周梳理1个工段工艺笔记）、现场实操任务（定时巡检、记录装置温度压力流量等数据、设备润滑、滤网清理、取样等辅助作业）、文档整理任务（运行日志、交接班记录、实习周记、月度小结）、合规任务（保密制度、禁烟、动火受限等现场管理规定）。
实习周志每周1篇，内容包含岗位操作、工艺学习、问题反思、安全心得。另需按月整理月度实习小结。
实习总结报告不少于3000字。包含：实习企业概况、实习工段工艺流程、岗位操作内容、技能收获、行业认知及职业规划。
时间安排第1周安全集训 → 第2-5周岗位认知 → 第6-18周岗位实习（全程在师傅监护下开展）。
考核方式安全考核30分 ＋ 实操考核40分（企业师傅打分，独立完成巡检记录、设备点检为达标）＋ 材料考核20分 ＋ 考勤素养10分。违规操作安全考核直接判为不及格。
⚠ 重要提醒
  • 所有学生实习前必须签订附件3：三方协议，未签协议不予认定实习。签订后按市域（区内）/省域（跨省）分开提交三方协议扫描件。
  • 实习期间在"i鄂院"上打卡。
  • 附件11：学生实习管理制度和附件12：学生实习安全管理规定为实习期间需遵守的管理制度（校企共同制定，需企业盖章）。
  • 严格通过企业三级安全教育（厂级、车间级、班组级）方可上岗。遵守操作规程，杜绝"三违"行为（违章操作、违章作业、违反劳动纪律）。
  • 实习期间不得擅自离岗、串岗，未经校企双方同意不得擅自终止实习。中途擅自调换实习单位，实习成绩为零分。请假须按校企流程审批。
  • 防爆区域严禁携带手机。不得乘坐摩托车、高速电瓶车。必须在实习单位指定的地方统一住宿。
  • 经常向班主任如实报告实习情况，不得谎报。转岗或新增工作内容要先向班主任报告。
  • 实习材料包括：三方协议、实习方案、学生实习报告、实习考核结果、实习日志（周志）、实习检查记录、实习总结、有关佐证材料。
